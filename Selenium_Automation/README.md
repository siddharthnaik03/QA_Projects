# OrangeHRM-Automation-Testing

## Project Overview
This project is a Behavior-Driven Development (BDD) automation framework designed to test the OrangeHRM web application using Selenium WebDriver, Cucumber, and TestNG.
The framework automates real user workflows such as login validation and employee creation while following industry-standard automation design practices like Page Object Model (POM).
The integration of TestNG enhances execution control, scalability, and reporting capabilities.
